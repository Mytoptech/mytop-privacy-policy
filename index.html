Privacy Policy

Last updated: March 2025

MyTop, developed by Antonio Carmona Polanco, collects certain personal data to provide a better user experience.

Data collected

  • Name
  • Email
  • Age
  • Country
  • Password (only if not using Google Sign-In)

Use of data

The data is used exclusively to display user profiles, generate statistics, and participate in global beauty contest rankings. No data is shared with third parties.

Google Sign-In

If the user signs in with Google, basic information from their Google account will be used to create their user profile within the app.

Advertisements

The app may display Google AdMob ads. These ads may use anonymous identifiers to personalize the advertising experience.

Security

All user data is securely stored using Firebase Authentication and Cloud Firestore services.

User rights

Users can delete their account or log out from the app at any time.
